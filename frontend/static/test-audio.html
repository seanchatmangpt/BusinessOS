Troubleshooting Steps:

  1. Check system volume - Make sure your Mac volume is up
  2. Check browser tab - Look for a mute icon on the browser tab itself
  3. Check output device - System Settings → Sound → Output (make sure correct speakers selected)
  4. Try headphones - Plug in headphones to rule out speaker issues
  5. Try different browser - Test in Safari or Firefox
  6. Check browser permissions - Site Settings → Sound (make sure not blocked)
