### 2. Development Workflow

- Use `.gitignore` to prevent logs from being committed
- Avoid creating backup files with `.backup` or `.bak` extensions (use git instead)
- Delete empty directories immediately
- Use meaningful branch names instead of numbered copies

### 3. Periodic Cleanup

Schedule quarterly cleanups to:
- Remove stale log files
- Check for duplicate directories
- Verify no backup files are committed
- Review and archive old test scripts
