## Best Practices

1. **Timeless Content**: Only add docs that won't become outdated
2. **Well-Organized**: Use subdirectories for related content
3. **Cross-Reference**: Link between related docs
4. **Keep Updated**: Update when underlying systems change
5. **Version ADRs**: Use ADRs for architectural changes
