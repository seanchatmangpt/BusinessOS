make voice-test-quick

# Clean test results
make voice-test-clean

Test Coverage:
--------------

STT Testing:
  - Basic speech recognition
  - Long utterances
  - Noisy audio
  - Accent variations
  - Speech rate variations

Agent Testing:
  - Greeting responses
  - Question answering
  - Command execution
  - Response quality
  - Latency validation

TTS Testing:
  - Audio generation
  - Quality metrics
  - Latency measurement

E2E Testing:
  - Full pipeline validation
  - End-to-end latency
  - Overall quality gates
  - Pass/fail criteria


Metrics Captured:
-----------------

Latency:
  - STT Latency: audio → transcription
  - Agent Latency: transcription → response
  - TTS Latency: response → audio
  - E2E Latency: total pipeline time

Quality:
  - STT Accuracy: word-level comparison
  - Word Error Rate: STT quality metric
  - Response Quality: agent output validation
  - Audio Quality: sample rate, bit depth

Thresholds:
  - STT: <1000ms, >90% accuracy
  - Agent: <3000ms, relevant response
  - TTS: <2000ms, quality audio
  - E2E: <6000ms, >75% pass rate
